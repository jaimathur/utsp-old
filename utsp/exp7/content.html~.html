Mathematical Morphology

Back To Experiment List

Mathematical morphology (MM) is the study of image processing methods based on the shape or form of objects. Mathematical morphology provides the ability to probe images using likely shapes of objects expected in the image and manipulate images using these probes.

Structuring element (SE)

These are the small set to probe the image under study. For each SE, origin must be defined. Its Shape and size must be adapted to geometric properties for the objects.

 

 

Idea of Structuring Element

Probe at each pixel in the binary image: Check if SE is ”satisfied”. Output pixel is set to 0 or 1 depending on used operation.

Points to Remember

Erosion

Removal of structures of certain shape and size, given by SE.

Dilation

Filling of holes of certain shape and size, given by SE.

Opening

Opening is defined as erosion followed by dilation. It can be expressed as :

Opening eliminates protrusions, breaks thin joints and smoothens contour.

Meaning of Opening

Treat the structuring element as a rolling ball. Boundary of A∘B = points in B that reach farthest into A when B is rolled inside A.

Closing

Closing is defined as dilation followed by erosion, denoted by •

Closing smoothens contour, fuses narrow breaks and long thin gulfs, eliminates small holes and fills gaps in the contour.

Meaning of Closing

See B as a ”rolling ball”. Boundary of A•B = points in B that reach farthest into A when B is rolled outside A.

Interpretation of dilation

After dilation you can easily notice that the white component in the image is increased in shape and size. i.e., black portion in the image is reduced. You can easily get this if you will note the black spots in the image. During dilation in gray scale image what happens is for each pixel, all image pixels coming under structuring element are considered and the maximum pixel value is assigned as the central pixel value. So only for those pixels where the complete structuring element lies in pure black the output is black.

MORPHOLOGICAL OPERATIONS



The image shown above is red band of sample image quickbird.jpg. Use this image for comparison in all the morphological operations discussed here.

EROSION



Observe that there are many number of black colored areas. All these areas have been removed by the structural element. By definition an erosion operation results in the removal of certain shape and size given by SE. The SE used here is square of size 3.

DILATION



Dilation is opposite to erosion operation. Observe that edges and many areas have been filled up. Looks like have grown in size. By definition a dilation operation fills holes of certain shape and size given by SE. In dilation operation disconnected edges get connected and they also grow thick in width.

OPENING



You might have noticed that above image looks like it is smoothened. This is because on implementing Opening operation, whatever the protrusions are present in the image, they are eliminated. It also breaks thin joints for example a weak link between edges. As a result we get a smooth contour. Observe the edges here, they are smooth than in the original image where edges are sharper.

CLOSING



Closing operation is somewhat similar in its result when compared to opening. Because as you can see the edges have been smoothened and whatever narrow breaks between edges were present they are now connected. This operation also eliminates holes and fills gaps in the contour (edges).

TOP-HAT FILTERING



Observe that all the bright (respectively dark) objects with uneven background are highlighted.

FAQs

Q 1. What does user requires to have to be able to use virtual satellite image processing and analysis lab?

Ans : Virtual Satellite image processing and analysis lab setup only requires internet connection and a web browser at the student's end to connect to the lab. The lab is self-contained, with sample images, aim, theory procedure, actual experiment and interpretation of the results. It is highly recommended to use v-SIPAL in mozilla firefox or google chrome and have adobe flash player installed in you PC.

Q 2. Which image formats are supported?

Ans : v-SIPAL supports all basic image formats like JPEG, BMP, TIFF, geoTIFF. In addition to these, raw satellite images can also be used with header file having an extension (.hdr)

Q 3. Are sample images there for users?

Ans : Each user is provided with two sample images : Sample 1 is a raw satellite image with 6 bands, Sample 2 is a jpeg file with 3 bands.

Q 4. Is there any restriction on image size to upload?

Ans : One can upload images of any size. But it is highly recommended to not upload images having dimensions more than 2048x2048.

Q 5. Can user download the output images?

Ans : Yes. User can download output images and histograms by just right clicking on them and then clicking on "Save image as".
