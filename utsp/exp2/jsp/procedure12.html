Procedure -  Neural Network Based Classification
 
These are the general guide lines and steps that should be followed to perform the experiment :
 
Step 1 : Select image on which you want to perform the NN based Classification.
You can also upload your own images by clicking on "upload more images".

Note : Neural network based classification will only work with raw satellite images with header in the format of .hdr

Step 2 : Select color composite to be used for display purpose.

Step 3 : Specify no. of bands to be used for classification
Step 4 : If you are not using all the bands for classification, then input the sequence of bands, separating them using space.
Step 5 : Input the following Network configuration parameters according to their limits as :

Dimension of feature space (>=1)

No. of NN hidden layers (>=1)

Learning Rate (e.g., 1)

Momentum Factor (e.g., 0.1)

Allowable Error (e.g., 0.001)

Normalization Factor (e.g., 1)

Maximum no. of iterations (e.g., 10000)

Step 6 : Enter name of the binary classified output image

This will give you the option to use this classified image further in other experiments like accuracy assessment.

Note: Do not enter the extension of file, just enter the name.
Step 7 : Click on "Display" button

After clicking on display button, it will show you the original image in the same window, below the experiment. Use this image to pick the training windows.
Step 8 : Enter Sequence for no. of nodes in each hidden layer.

Note : Follow the limitation for this value as written in the experiment itself.

Step 9 : Pick the training window coordinates and color

Before picking training window coordinates you must pick the color for the class. You can pick it by clicking on "Color picker". To pick a training window, click on the button "Pick U" to input the coordinates of upper left corner of window and "Pick L" to input the lower right corner of window.

Step 10 : To add more training area for same class, click on "More training area" button

Step 11 : To add more class, click on "Next class" button

Step 12 : Click on "Process" button

This will give you the desired classified image and it will place the binary version of classified image in your account folder with the name you provided.
