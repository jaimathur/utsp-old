Procedure - Mathematical Morphology
 
These are the general guide lines and steps that should be followed to perform the experiment :
 
Step 1 : Select Image

Select the image on which you want to perform the experiment.
You can also upload your own images by clicking on "upload more images".
Step 2 : Select band

Morphological operations will be applied on the selected band.
 
Step 3 : Select operation

You can select any operation that you want to perform.
For the operation "Boundary Detection", you will have to select the type of boundary detection algorithm as :

And if you choose "Hit or Miss Transform", then you will have to enter the dimensions of both the structuring element. i.e., dimensions of SE for object extraction and dimensions of SE for background extraction, as:

then click on "Define SE" button to select the active elements of structuring element.

Step 4 : For any operation, except "Hit or Miss Transform", you can choose inbuilt SE or you can define your own SE by selecting "custom" in "Select Structuring Element" option as :

If you have chosen "custom" in "select structuring element" option, then click on "define SE" button after entering the width and height of SE. It will ask to mark the elements that you want to be active in your SE as :
Step 5 : Click on the button "Process"

This will give you the Original image and output of morphological operation.
