Procedure - Discrete 2D Fourier Transform
 
These are the general guide lines and steps that should be followed to perform the experiment :
 
Step 1 : Select Image

Select the image on which you want to perform the experiment.
You can also upload your own images by clicking on "upload more images".
Step 2 : Select band combination for displaying the image..

As we know that an image can have more than three bands, so we need to specify which bands should be used for display purpose..
 
Step 3 : Select transforming operation

You can select any operation that you want to perform. "Spectrum Only" will give you the Fourier Magnitude of original image. Other two filters are provided to show the utility of fourier transform in image processing. For more details you can read the theory of experiment.

For filtering you will have to specify whether you want low pass or high pass filtering. So, select low pass or high pass from the option "Band".
Step 4 : Select the specific band for which you want to see the fourier transform output.
 
Step 5 : Enter other parameters specific to filters and then click on the button "Process"

This will give you the Original image and output of filtering operation. For more details regarding the filter specific parameters, read the theory of experiment.
