Theory

 

Discrete 2D Fourier Tansform

Back To Experiment List

A transform, including the Fourier Transform (FT), is essentially a mathematical mapping process. A transform maps image data into a different mathematical space via a transformation equation. Most of the discrete transforms map the image data from the spatial domain to the frequency domain, where all the pixels in the input (spatial domain) contribute to each value in the output (frequency domain). Rapidly changing brightness values over short distances correspond to high frequency terms, slowly changing brightness values correspond to low frequency terms.

 

 

Display of Fourier Transform

The Fourier transform consists of complex floating point numbers, stored with a data type of complex. For convenience of viewing, we shift the origin to the center of the image by applying the properties of periodicity and translation with u0 = M/2 and v0 = N/2.

Advantages of Centering Fourier Transform Display

  • Easier to interpret the spectral information with the origin in the center and frequency increasing radially outward from the center
  • Makes it easier to visualize the filtering in the frequency domain
  • Visually more appealing

Interpretation from spectrum The spectrums shown to you as outputs are actually transformed spectrums. The (0,0) in original spectrum lies at top left corner while for better understanding it is transformed to the centre of spectrum.

High frequency components of image

The centre of the spectrum shows the lowest frequency component in the original image and increases radially away from the centre.

Low frequency components of image

Points away from the centre shows the high frequency components in the original image.

Edges

If you are able to see some linear features in the spectrum then those are revealing edge components in image and there directions shows the major edge directions in image.

How to perform filtering?

Using any filter which can operate in frequency domain like Gaussian and Butterworth filter (see theory for details), can be used to perform two basic filtering operations : Low pass filtering and High pass filtering.

Low pass filtering

Low pass filtering using spectrum is done as :

First we choose the low pass band according to the parameters specific to applied method (Gaussian or Butterworth)

Then we have to take the inverse fourier transform for the output of first step and we get the desired low pass filtered image

High pass filtering

High pass filtering using spectrum is done as :

First we choose the high pass band according to the parameters specific to applied method (Gaussian or Butterworth)

Then we have to take the inverse fourier transform for the output of first step and we get the desired high pass filtered image.

Gaussian filtering in frequency domain

The first figure represents the original image (quickbird.jpg) while second is the lowpass filtered image. Observe the blurring in areas enclosed in oval figures by comparing both images. All the high frequency components that fall beyond the radius specified (sigma1) are attenuated and low frequency components are kept in tact.



The figure above represents Gaussian high pass filtering in frequency domain. The edges are associated with high frequency components. Similarly there are other areas where one would find abrupt changes in gray levels. All such areas correspond to high frequency components. In high pass filtering the low frequency components which fall within a radius specified by sigma are cut off from the image. Hence only high frequency components remain. Observe that edges and other areas come up in the above image.

Butterworth filtering in frequency domain



The image shown above is butterworth lowpass filtered image of quickbird.jpg sample image. Observe the blurring effect the lowpass filtering has produced. This filtering was done with sigma1=2 and sigma2=2. You should note in lowpass filtering the high frequency components are suppressed. Hence the edges, which belong to high frequency components are blurred or made less sharp.



The image shown above is the result of high pass butterworth filtering. All the low frequency components are suppressed here. Edges and other regions associated with high frequency components are improved.

FAQs

Q 1. What does user requires to have to be able to use virtual satellite image processing and analysis lab?

Ans : Virtual Satellite image processing and analysis lab setup only requires internet connection and a web browser at the student's end to connect to the lab. The lab is self-contained, with sample images, aim, theory procedure, actual experiment and interpretation of the results. It is highly recommended to use v-SIPAL in mozilla firefox or google chrome and have adobe flash player installed in you PC.

Q 2. Which image formats are supported?

Ans : v-SIPAL supports all basic image formats like JPEG, BMP, TIFF, geoTIFF. In addition to these, raw satellite images can also be used with header file having an extension (.hdr)

Q 3. Are sample images there for users?

Ans : Each user is provided with two sample images : Sample 1 is a raw satellite image with 6 bands, Sample 2 is a jpeg file with 3 bands.

Q 4. Is there any restriction on image size to upload?

Ans : One can upload images of any size. But it is highly recommended to not upload images having dimensions more than 2048x2048.

Q 5. Can user download the output images?

Ans : Yes. User can download output images and histograms by just right clicking on them and then clicking on "Save image as".
