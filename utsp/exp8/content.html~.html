Motivation:

In a given pair of bands the response of two objects is generally different. For example in the diagram below you can see the different response of different objects in different wavebands.

 

 

Pixel by pixel comparison between images can highlight pixels that have very high difference in reflectance in such bands. Operations like band difference and band ratio or combinations of them are popularly used for this purpose. Some of such operations are described below :

Band Ratio

Band ratio is a very common operation. For each pixel with position (m,n) can be expressed as :

Ratioi,j(m,n) = Bandi (m,n) / Bandj(m,n)

Here, m and n are row and column position respectively. i and j denotes the bands over which operation is performed. If Bandj(m,n) = 0, suitable adjustment has to be made (e.g., add +1 to the denom.). Minimum ratio will be 0; Maximum ratio will be 255.

Normalized Difference Vegetation Index (NDVI)

Combination of band differences, ratios and additions can result in useful outputs that can highlight features like green vegetation. One such feature is Normalized Difference Vegetation Index (NDVI). It can be expressed as :

NDVI(m,n) = [Band IR (m, n) − Band R (m, n)] / [Band IR (m, n) + Band R (m, n)]

NDVI results in high values where IR dominates red wavelength. This happens where vegetation is present. Range of NDVI is [-1 +1]. NDVI has been widely used in a wide range of agricultural, forestry and biomass estimation applications. It is also used to measure the length of crop growth and dry-down periods by comparing NDVI computed from multidate images.

Soil Adjusted Vegetation Index (SAVI)

The soil adjusted vegetation index (SAVI) introduces a soil calibration factor, L, to the NDVI equation to minimize soil background influences resulting from first order soil-plant spectral interactions. It can be expressed as :

SAVI =[(1 + L ) * ( NIR − red )] / [NIR + red + L]

Atmospheric Adjusted Vegetation Index (ARVI)

The atmospheric effects are accounted for in ARVI. It can be expressed as :

ARVI = [(p * nir − p * rb)] / [(p * nir + p * rb)]

Where, p*rb = p*red −γ ( p*blue − p*red)

p* indicates the atmospherically corrected versions of NIR, Red and Blue bands for molecular scattering and ozone absorption.

Enhanced Vegetation Index (EVI)

EVI is a mixture of SAVI and ARVI, in that both atmospheric effects and soil effects are accounted for. It can be expressed as :

EVI = [(p * nir − p * red)] / [(p * nir + C1 p * red − C2 p * blue + L)]

Where, C1 and C2 describe the use of the blue band in correction of the red bandfor atmospheric aerosol scattering. The coefficients, C1, C2, and L are empirically determined as 6.0, 7.5, and 1.0, respectively for MODIS. This algorithm has improved sensitivity to high biomass regions and improved vegetation monitoring through a de-coupling of the canopy background signal and a reduction in atmospheric influences.

Interpretation of NDVI

NDVI is calculated at each pixel NDVI(m,n) as:

NDVI(m,n) = [IR (m, n) − Red (m, n)] / [IR (m, n) + Red (m, n)]

where m and n are row and column position of pixel in the image, IR and Red are infrared band and red band in the image respectively.

Why Red and Infrared band?

Normalized difference vegetation index sets the pixel values to peek for pixels corresponding to high vegetation. From the graph below one can can notice that for vegetation the spectral response is very high in near infrared region while it is very low in red wave band. So the difference in near infrared and red band is highest for the pixels corresponding to high vegetation area. That is the reason we are performing (IR-Red) in the numerator of NDVI. The denomenator (IR + Red) is used to normalize the values of (IR-Red) to have the NDVI values in the range of -1 to +1.



For displaying the NDVI result as an image scaling of NDVI values has to be done so that pixel values will range from 0 to 255 according to their NDVI values.

From the output NDVI image for our experiment you can easily say that the brightest regions are region of high vegetation and as the brightness fades out the strength of vegetation is decreases.

Simple ratio



You can easily make out that the vegetation areas are highlighted here. These areas are more brighter than other areas. This is one of the use of simple ratioing of two bands to highlight the large difference in corresponding gray values. All the white colored areas represent vegetation. Also the health of the vegetation can be inferred from the brightness levels. For example more bright areas represent very healthy vegetation and this health decreases as the brightness values decrease.

Square Root



Comparatively the square root method highlights vegetation areas more outstandingly than the previous method. All the white colored areas represent vegetation. Also the health of the vegetation can be inferred from the brightness levels. For example more bright areas represent very healthy vegetation and this health decreases as the brightness values decrease.

Normalised Difference Vegetation Index

It is another technique for studying vegetation over a certain area. It gives much better results than ratio or square root methods. An NDVI image is shown below. Water content in vegetation can estimated by NDWI. Vegetation with lower water content is brighter compared to others.

FAQs

Q 1. What does user requires to have to be able to use virtual satellite image processing and analysis lab?

Ans : Virtual Satellite image processing and analysis lab setup only requires internet connection and a web browser at the student's end to connect to the lab. The lab is self-contained, with sample images, aim, theory procedure, actual experiment and interpretation of the results. It is highly recommended to use v-SIPAL in mozilla firefox or google chrome and have adobe flash player installed in you PC.

Q 2. Which image formats are supported?

Ans : v-SIPAL supports all basic image formats like JPEG, BMP, TIFF, geoTIFF. In addition to these, raw satellite images can also be used with header file having an extension (.hdr)

Q 3. Are sample images there for users?

Ans : Each user is provided with two sample images : Sample 1 is a raw satellite image with 6 bands, Sample 2 is a jpeg file with 3 bands.

Q 4. Is there any restriction on image size to upload?

Ans : One can upload images of any size. But it is highly recommended to not upload images having dimensions more than 2048x2048.

Q 5. Can user download the output images?

Ans : Yes. User can download output images and histograms by just right clicking on them and then clicking on "Save image as".
