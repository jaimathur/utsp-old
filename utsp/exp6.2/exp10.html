Simulation of Control of Magnetic Levitation System
PID Control for desired transient response – Unstable Plant
Aim

In this experiment, the aim is to design a PID controller for the magnetic levitation system in an unstable configuration. The design specifications are the peak overshoot, settling time, and zero offset. We study the design technique, use the technique to design a PID controller, and validate the same through closed loop simulations. We also explore the effect of different PID settings on the transient response characteristics.

Introduction

In the previous experiment, with PD control, we could stabilize the unstable Maglev plant transfer function, and achieve the desired PO and ts. However, with a PD controller, we cannot achieve zero offset. By adding the I mode, we get the PID controller which eliminates offset.


In this experiment, we learn how to design a PID controller for the unstable configuration of the Maglev plant. As before, the design procedure is based on pole-zero cancellation, reducing the closed loop transfer function from third to second order. Then, the design is done using standard relations for a second order system. We validate the PID controller design through closed loop simulations. We also explore the effect of different PID settings on the transient response characteristics.

Things to think about

Things to think about:

2. Can you think of other real world applications where PID control is used to stabilize an unstable plant?

3. Can this unstable Maglev configuration be stabilized using PI control?

4. If we flip the magnet over, will this Maglev configuration be stable?

6. Is it possible to control both the magnets simultaneously using one actuator coil?
