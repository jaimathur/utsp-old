Procedure - Edge Detection
 
These are the general guide lines and steps that should be followed to perform the experiment :
 
Step 1 : Select Image

Select the image on which you want to perform the experiment.
You can also upload your own images by clicking on "upload more images".
Step 2 : Select band for which you want to get edge output.

Edge detector will give you edge output for the band you will select here.
 
Step 3 : Select method of edge detection

You can select any method of edge detection from the available options as :

Then enter threshold, sigma and direction values as required according to the method you select. For more details about these parameters refer to the theory of experiment.
Step 4 : Click on the button "Process"

This will give you the Original band image and output of edge detection operation.
