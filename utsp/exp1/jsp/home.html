Welcome to Virtual Labs

Satellite images need to be processed and analyzed to extract meaningful information from them. The inherent geographical reference in the satellite images helps take decisions that improve the condition of respective geographical areas. The images are enhanced, transformed and analyzed using mathematical and statistical techniques, that are implemented using popular software development tools and intuitive user interfaces allow the user to run the software, organized as experiments, and gain knowledge about the underlying scientific processes

Virtual Labs Setup

Virtual Satellite image processing and analysis lab setup only requires internet connection and a web browser at the student's end to connect to the lab. The lab is self-contained, with sample images, aim, theory, procedure, actual experiment and interpretation of the results

Satellite Image Processing and Analysis

Satellite imagery consist of images of Earth or other planets acquired by sensors mounted on earth orbiting satellites. Satellite images are used by practitioners of agriculture, geology, forestry, biodiversity conservation, regional planning, education, intelligence and warfare. Satellite images can be in visible, infrared, and other wavelengths. Satellite image processing and analysis provide qualitative as well as quantitative information dealing with natural resources management, environmental monitoring, disaster mitigation, rural and urban development applications
