Procedure -  Accuracy Assessment
 
These are the general guide lines and steps that should be followed to perform the experiment :
 
Step 1 : Select the classified binary image for which you want to perform accuracy assessment.
Step 2 : Select the corresponding image file for display purpose.

It will help you to pick testing window coordinates.

Step 3 : Select color composite to be used for display purpose.

Step 4 : Click on "Display" button

After clicking on display button, it will show you the original image in the same window, below the experiment. Use this image to pick the training windows.
Step 5 : Pick the testing window coordinates and color

Before picking testing window coordinates you must pick the color for the class. You can pick it by clicking on "Color picker". To pick a testing window, click on the button "Pick U" to input the coordinates of upper left corner of window and "Pick L" to input the lower right corner of window.

Step 6 : To add more test area for same class, click on "More test area" button
 
Step 7 : To add more class, click on "Next class" button

Step 8 : Click on "Process" button

This will give you the overall accuracy, producer's accuracy, user's accuracy and Kappa coefficient for the classified image as output.
