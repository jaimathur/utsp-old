Procedure -  K-Means Clustering
 
These are the general guide lines and steps that should be followed to perform the experiment :
 
Step 1 : Select Image

Select the image on which you want to perform the experiment.
You can also upload your own images by clicking on "upload more images".
Step 2 : select desired bands to be used for clustering.

Select "All", if you want to use all bands of the image. Otherwise, select "Custom" and then mark the bands to be used as :

Step 3 : Specify initial cluster centers.

If you want it to randomly select cluster centers automatically then choose "Random". Otherwise, select "User Specified". If you will choose "User Specified", then you will have to select a cluster center file (*.ctr) in the next option.

Note: (*.ctr) file should match according to the number of classes, bands and image you are going to use.
Step 4 : Enter other parameters

Enter the number of desired classes and name of the file for storing final cluster centers, generated after clustering by the operation.

Note : Do not provide any extension to the file name. It will automatically create the file with an extension (.ctr), so you would be able to use it further for specifying initial cluster centers.

Step 5 : Click on the button "Process"

This will give you the clustered image as output.
