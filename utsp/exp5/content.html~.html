What Is An Edge?

An edge is a set of connected pixels that lie on the boundary between two regions. The pixels on an edge are called edge points. Gray level / color / texture discontinuityacross an edge causes edge perception. Position & orientation of edge are key properties.

You can see various kind of edges due to various reasons, below:

 

Edges due to color difference

Edges due to intensity or illumination difference

Edges due to texture difference


Edges due to surface orientation difference

 

 

  Types of Edges

Basically edges are of three types : Step edges, Ramp Edges and Pulse/Peak edges.

Step Edge : Abrupt discontinuities such as the boundary between land and water ponds or tall objects and their shadows are instances of step edges.

Ramp Edge : It indicates a more gradual transition from one feature to another.

Pulse/Peak Edge : These are like thin ribbon like features that are long but they also have a finite width.

Interpretation Of Edge Detection Outputs

Here we will discuss how to interpret the outputs produced by experiment 5, i.e., Edge Detection. So, please follow the steps as shown below to get the sample output for interpretation. 1) Select "inputimage" as your experimental image. "inputimage" is a satellite image having 6 bands stored in BIL format as raw image. It has the dimension 801x1401. Since in our discussion we will cover upto its 4th band, so it is necessary to understand that what the first 4 bands of this image represents. The first four bands are as follows: blue, green, red and infra-red respectively. 2) Select band for which you want to find the edge output. Here, we are selecting band "5". 3) Select method of edge detection. Here we will choose "Sobel". (already set as default) Sobel operator uses two perpendicular masks for detecting horizontal and vertical edges. So, here you have an option to select whether you want only horizontal edges, vertical edges or both. Consequently, you will also have to enter threshold values seperately. 4) Enter "Edge threshold value" as "0.05", "Vertical Threshold" as "0.05" and "Horizontal threshold" as "0.05". Threshold limits the output to be displayed. By entering 0.05 as threshold you are actually limiting it to show only the magnitude and direction where it is above 5% of highest gray level that can be displayed, which is 255. 5) Click on "Process" button You will get the output as : Edge Detected Image

Laplacian of Gaussian (LoG) Threshold chosen to generate edge image below is 0.7 and sigma as 0.5. Laplacian filters are derivative filters and are prone to noise detection. Hence LoG operator smoothens the test image and then applies the laplacian operator. Hence the name LoG.



Canny operator Threshold =0.2 and sigma=0.5. Please note that in the experiment we enter lower threshold value. The program automatically calculates the higher threshold value (i.e twice the lower threshold). The image shown below represents edges as bright lines. The values that fall above higher threshold are labeled as edges. And those falling below lower threshold are discarded as not edges. Those falling in-between higher and lower are evaluated for nearness. Pixels with weak connectivity are not joined whereas pixels with higher chance of connectivity are connected.

FAQs

Q 1. What does user requires to have to be able to use virtual satellite image processing and analysis lab?

Ans : Virtual Satellite image processing and analysis lab setup only requires internet connection and a web browser at the student's end to connect to the lab. The lab is self-contained, with sample images, aim, theory procedure, actual experiment and interpretation of the results. It is highly recommended to use v-SIPAL in mozilla firefox or google chrome and have adobe flash player installed in you PC.

Q 2. Which image formats are supported?

Ans : v-SIPAL supports all basic image formats like JPEG, BMP, TIFF, geoTIFF. In addition to these, raw satellite images can also be used with header file having an extension (.hdr)

Q 3. Are sample images there for users?

Ans : Each user is provided with two sample images : Sample 1 is a raw satellite image with 6 bands, Sample 2 is a jpeg file with 3 bands.

Q 4. Is there any restriction on image size to upload?

Ans : One can upload images of any size. But it is highly recommended to not upload images having dimensions more than 2048x2048.

Q 5. Can user download the output images?

Ans : Yes. User can download output images and histograms by just right clicking on them and then clicking on "Save image as".
