Image Classification :

Assigning pixels in the image to categories into classes of interest is known as image classification.

Examples: built-up areas, water body, green vegetation, bare soil, rocky areas, cloud, shadow, etc.

Classification is done on the basis of features. Features are attributes of the data elements based on which the elements are assigned to various classes. E.g., in satellite remote sensing, the features are measurements made by sensors in different wavelengths of the electromagnetic spectrum – visible/infrared / microwave/texture features. etc.

K - Means Clustering :

It is an iterative algorithm for clustering. This is the most popular clustering algorithm. Number of Cluster 'K' is already known to user. Its steps are as follows:

Step 1 : K cluster mean vectors are initialized randomly.

Step 2 : Each pixel is assigned to one of the cluster according to the minimum feature distance from the cluster mean vectors.

Step 3 : After each pixel is assigned to some cluster, new mean vectors are computed for each cluster according to the assignment.

Step 4 : New cluster mean vectors are checked from the previous cluster mean vectors. If there is a difference greater than some threshold then this process is repeated until cluster means are stabilized.



Here, data heap is the set of all pixels. ui is the initial cluster mean vectors. Ci are clusters.

 

Here, for convergence of algorithm, new cluster mean vectors are compared with previous cluster mean vectors. Some important points to be noted :

Choice of K is important.

For normal remotely sensed images, about 15-20 iterations are maximum required for stabilization of the cluster centers.

Convergence can be based on average difference between cluster centres or maximum difference between any pair of cluster centers.

Interpretation Of K-means Clustering Outputs

Here we will discuss how to interpret the outputs produced by experiment 9, i.e., k-means clustering. So, please follow the steps as shown below to get the sample output for interpretation.

1) Select "inputimage" as your experimental image.

"inputimage" is a satellite image having 6 bands stored in BIL format as raw image. It has the dimension 801x1401. Since in our discussion we will cover upto its 4th band, so it is necessary to understand that what the first 4 bands of this image represents. The first four bands are as follows: blue, green, red and infra-red respectively.

2) Select "No. of bands to be used" as "All"

Number of bands actually works as number of feature vectors. So here we are using all the bands for clustering. Instead of selecting "All" one can also experiment with "custom" and then can select particular bands to be used for clustering

3) Enter "Maximum number of iteration" as "5"

K-means clustering has two terminating conditions : first, if successive iterations does not produces significant difference in clustered output. Second, if number of iterations exceeds maximum number of iterations specified by user. The output will be better if you increase this number. One can always experiment with these values.

Note : If you are using a large size image, it will be better to keep this number less to not overload the server.

4) Select "Initial cluster center" as "Random"

If you are clustering an image using k-means clustering then for first time you need to specify the starting cluster centers for each class. So, one can always choose "Random" to generate the initial clusters randomly. But, if you are going to perform k-means clustering on same image that you have used previously for clustering with same number of classes ythen you can use the final cluster centres produced by the program as initial cluster centers. So, it will be like iterating the clustering on same image further.

5) Enter "Desired no. of classes" as "3"

This will depend on your need that in how many classes you want to cluster the image. For testing, we are taking it as 3.

6) Enter any name in "Name of file, storing final cluster centers", Here we are entering the name as "inputimage_centre_5_3"

Keeping the name in this format will help you further to recognize that this cluster center file is having final clusters for "inputimage" with max. no of iterations "5" and number of classes as "3". The extension ".ctr" will be added automatically by the program.

7) Click on "Process" button

You will get following image as output :

K-means clustered image of inputimage



Here one can easily notice that the original image is clustered into three regions with three different colors. The colors are chosen randomly by the program. This is the reason that why we need to perform clustering. Since, during clustering it uses the different bands as feature vector, the clustering output classify efficiently according to the objects present in the image. As you have already seen the FCC of this image and also interpreted that, so you can easily identify that the regions in this clustered image as water body (green), vegetation (yellow), built-up areas or other (purple).

FAQs

Q 1. What does user requires to have to be able to use virtual satellite image processing and analysis lab?

Ans : Virtual Satellite image processing and analysis lab setup only requires internet connection and a web browser at the student's end to connect to the lab. The lab is self-contained, with sample images, aim, theory procedure, actual experiment and interpretation of the results. It is highly recommended to use v-SIPAL in mozilla firefox or google chrome and have adobe flash player installed in you PC.

Q 2. Which image formats are supported?

Ans : v-SIPAL supports all basic image formats like JPEG, BMP, TIFF, geoTIFF. In addition to these, raw satellite images can also be used with header file having an extension (.hdr)

Q 3. Are sample images there for users?

Ans : Each user is provided with two sample images : Sample 1 is a raw satellite image with 6 bands, Sample 2 is a jpeg file with 3 bands.

Q 4. Is there any restriction on image size to upload?

Ans : One can upload images of any size. But it is highly recommended to not upload images having dimensions more than 2048x2048.

Q 5. Can user download the output images?

Ans : Yes. User can download output images and histograms by just right clicking on them and then clicking on "Save image as".
